# 🐱 KittyFocus

KittyFocus 是一款结合 **番茄钟（Pomodoro）** 与 **虚拟宠物养成** 的开源专注工具。项目通过可爱的小猫陪伴动画提升用户专注体验，并利用情感反馈机制帮助用户养成良好的时间管理和自律习惯。

---

## 📖 项目简介

传统番茄钟应用虽然能够帮助用户管理时间，但趣味性较低，用户容易中途放弃。KittyFocus 在番茄钟的基础上加入了小猫陪伴机制：

- 专注成功，小猫健康成长。
- 中途放弃或专注失败，小猫会减少生命值，生命值耗尽后小猫死亡。
- 用户可以通过持续专注陪伴小猫成长，培养长期自律习惯。

---

## ✨ 功能特点

- ⏱️ 番茄钟专注计时
- 🐱 小猫陪伴动画
- ❤️ 小猫生命值系统
- 💀 专注失败减命及死亡反馈
- 📊 专注记录统计
- 🎨 简洁易用的用户界面

## 🚀 使用方法

1. 克隆项目

```bash
git clone https://github.com/Bistu-OSSDT-2026/KittyFocus.git
```

2. 进入项目目录

```bash
cd KittyFocus
```

3. 使用浏览器打开项目（或启动本地服务器）

开始设置专注时间即可体验 KittyFocus。

## 📌 项目开发流程

项目采用 GitHub 协作开发流程：

1. 创建 Issue 拆分开发任务。
2. 每位成员创建独立分支开发。
3. 完成功能后提交 Pull Request。
4. 由其他成员进行 Code Review 和测试。
5. 审核通过后合并至 main 分支。
